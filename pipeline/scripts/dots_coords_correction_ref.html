Dots coords correction¶

dots_coords_correction.dots_coords_correction()[source]¶

This script is used to collect all the raw countings from the different hybridization, correct the coords according to the registration of the reference gene and remove the dots that overlap in the overlapping regions between the images.

Input via argparse:

path – string. Exact path to the experiment folder pxl – int radius of pixel used to create the nhood

that is used to define that two pixels are the same

Returns: Save the aggregate coords and also the coords after dots processing
