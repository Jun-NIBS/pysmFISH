Staining segmentation¶

staining_segmentation.staining_segmentation()[source]¶

This script will segment the selected staining and output the identified objects.

All the parameters are entered via argparse.

scheduler: string
tcp address of the dask.distributed scheduler (ex. tcp://192.168.0.4:7003). default = False. If False the process will run on the local computer using nCPUs-1
path: string
Path to the processing directory
processing_file: string
Path to the hdf5 file with the staning to process
segmentation_staining: string
Staining to be segmented
